# Project Name Complete time
1 AcadMap 2020 (year 3)
AcadMap is a forum implemented by PHP and MySQL database. This is a static web page for users to get professional help on their future and studies. This is the product in group project in CSCI3100 Software Engineering.
2 Hospital Lookup 2021 (year 4)
Hospital Lookup is a single page application (SPA) implemented by React.JS and Node.JS. This project use MongoDB as the database (which is non-relational) and apply multiple API such as Google Map API and data gov info API. This is the product in group project in CSCI2720 Building Web Applications.
3 Unity Game: Cthulhu Party(Name subject to change) 2021 (year 4)
Cthulhu Party is a 2D multiplayer RPG(role-playing game) developed by Unity. This project adapt a existed TRPG script as the base script of the game. My group implement the game by Unity with suitable modification of script and adding game mechanics. A client-server model implemented by Mirror is applied to support multiplayer.
